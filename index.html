🚀 快速接入

只需在页面底部添加一行脚本标签，即可自动连接 AI 对话：

<script src="chat-widget.js"></script>

或者使用自定义配置：

<script src="chat-widget.js"></script>
<script>
  AIChatWidget.init({
    primaryColor: '#667eea',
    title: '智能助手',
    placeholder: '请输入您的问题...',
    welcomeMessage: '你好！我是 AI 助手，有什么可以帮你的？',
    position: 'right',
    botId: 'your-bot-id',
    accessKeyId: 'your-access-key-id',
    accessKeySecret: 'your-access-key-secret'
  });
</script>

📋 配置项说明

primaryColor - 主题色，默认 #667eea

title - 聊天窗口标题，默认 "AI 助手"

placeholder - 输入框占位文字

welcomeMessage - 欢迎消息

position - 悬浮球位置，left 或 right

botId - 机器人 ID（从管理后台获取）

accessKeyId - API 鉴权密钥 ID

accessKeySecret - API 鉴权密钥

botAvatar - 机器人头像 URL
